# ZHAW LSFM Thesis Template Documentation

This repository contains the documentation website for the ZHAW LSFM Quarto-Typst thesis template.

## Documentation Website

Visit the documentation at: <https://zhaw-lsfm.github.io/quarto-thesis-docs>

## Template Repository

The actual template is hosted at: <https://github.com/zhaw-lsfm/quarto-thesis>

## Installation

To install the template:

```bash
quarto use template zhaw-lsfm/quarto-thesis
```

## Contributing

See [For Developers](https://zhaw-lsfm.github.io/quarto-thesis-docs/for-devs.html) for contribution guidelines.

## Local Development

```bash
# Clone this repository
git clone https://github.com/zhaw-lsfm/quarto-thesis-docs.git
cd quarto-thesis-docs

# Preview locally
quarto preview

# Publish to GitHub Pages
quarto publish gh-pages
```
